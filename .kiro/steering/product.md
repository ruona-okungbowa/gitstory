# Product Overview

GitStory is an AI-powered career readiness platform that transforms GitHub repositories into interview-ready portfolio materials.

## Core Features

- **Portfolio Analysis**: Analyzes GitHub repos and generates a 0-100 portfolio score
- **AI Content Generation**: Creates STAR stories, resume bullets, and professional READMEs
- **Career Guidance**: Identifies skill gaps, recommends projects, and provides personalized learning paths
- **Interview Prep**: Mock interview simulator with AI-powered feedback
- **Job Matching**: Matches portfolios to job descriptions

## Target Users

Developers preparing for job interviews who need to showcase their GitHub projects professionally and identify areas for improvement.

## Key Integrations

- GitHub OAuth for repository access
- OpenAI GPT-4 for content generation and analysis
- Supabase for authentication and data persistence
